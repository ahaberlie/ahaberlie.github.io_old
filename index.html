Code Examples

To implement a method proposed by this paper, I needed to write a point rotation procedure from scratch This method was utilized for an upcoming research article

Point rotation

Using Unidata's pyUDL module, I wrote a script that uses downloaded national radar composites from Unidata and generates a hovmoller diagram for a recent time period that can be modified by the user. Uses numpy and matplot lib. A possible alternative for those who don't have IRIS. Code to download Unidata's national mosaic was borrowed from Unidata's IPython Examples

Radar Hovmoller with pyUDL and numpy

Using SPC's tornado dataset, I wrote a script to generate a hovmoller diagram using tornado touchdown locations. The output is a neat way to visualize tornado occurrence in the United States for every hour of the day. Uses numpy, matplotlib, and pandas. Data from SPC's svrgis page: http://www.spc.noaa.gov/gis/svrgis/

Tornado touchdown Hovmoller with pandas and numpy

Noticed that by default, pixels of low or noise values in radar images (and other sparse datasets) displayed over top of images or lines on basemap. I created this example to show an application of setting low values to none. looks better than just having the lowest colormap value overlayed. Radar files are downloaded remotely via pyUDL from Unidata using examples from: Unidata's IPython Examples.
